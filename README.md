## Stack

    - React
    - Typescript
    - Material UI
    - Styled Component  

    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd pokemon-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run cypress:open
  npm run cypress:run
```
